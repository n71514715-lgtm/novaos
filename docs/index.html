Install on Android

One command. Any Android 7+ phone. No root required.

curl -sL https://raw.githubusercontent.com/n71514715-lgtm/novaos/main/termux/nova-quick-install.sh | bash

Requires Termux from F-Droid · Android 7.0+ · No root · No accounts · No telemetry



Clone and build

git clone https://github.com/n71514715-lgtm/novaos && cd novaos && nova setup

Why NovaOS exists.

Because your OS shouldn't spy on you. Windows sends telemetry. Android reports to Google. macOS checks your apps with Apple. Every major OS today is a surveillance tool dressed as a product.

Because AI shouldn't require surrendering your data. NovaBrain runs entirely on your device. Your questions never leave. The model sits on your SSD. The cloud never sees a byte.

Because privacy tools shouldn't require a PhD. One command. One wizard. Anyone can use NovaOS — the journalist in Tehran, the activist in Belgrade, the student in Dhaka.

Because the people who need privacy the most can least afford it. NovaOS is free. Open source. Runs on a $30 Android phone.
